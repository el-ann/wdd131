Ghanaian Hair Industry Website Project Plan

Site Name

Ghanaian Hair Industry

This name reflects the focus of the site on celebrating the Ghanaian hair industry, showcasing braids, wigs, and natural hair care, as well as supporting local stylists and brands.

Site Purpose

The site provides a platform to celebrate and promote the Ghanaian hair industry, featuring profiles of local stylists, galleries of traditional and contemporary hairstyles, hair care tips, and links to local e-commerce stores for hair products. The website will educate users about hair styling techniques, provide resources, and offer a space to buy authentic Ghanaian hair products.

Scenarios

  • What are the best styling techniques for braids and wigs that work with my natural hair type?
  • Where can I purchase high-quality Ghanaian hair products online?

Color Schema

The website will use the following colors:

  • Primary Color: #f39c12 (Yellow-orange) - Used for buttons, links, and accents.
  • Secondary Color: #16a085 (Teal) - Used for background, navigation bar, and headings.

Typography

The following fonts will be used on the site:

  • Headings: "Poppins", sans-serif (Large, bold font used for titles and headings)
  • Body Text: "Roboto", sans-serif (Easy-to-read font for general text)

Testing

The website will undergo the following tests:

  • Use of browser developer tools to check for errors and validate HTML/CSS.
  • Running a Lighthouse report to test performance, accessibility, and SEO.
  • Color contrast checks using the DevTools CSS overview.
  • Testing JavaScript functionality for dynamic elements like the product cart and form submissions.
